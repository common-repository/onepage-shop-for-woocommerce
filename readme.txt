== Installation ==

1) Copy/Upload 'gen-woo-onepage-shop' folder to the '/wp-content/plugins/' directory
2) Activate 'WooCommerce OnePage Shop' from wp plugin option in admin area
3) Plugin will appear in the menu bar of WP Dashboard
4) create a page and then call shortcode: [gen_wops_onepage_shop]  to generate the front end One Page Shop for woocommerce.
